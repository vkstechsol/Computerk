Improve your Computer Knowledge-

H-1.What is a computer?

A computer is an electronic device that can process and store data. "

2. What are the main components of a computer?

The main components of a computer are the central processing unit (CPU), the motherboard, the memory (RAM), the hard drive, and the power supply.

3. What is a mouse?

A mouse is a device used to control the cursor on a computer screen.

4. What is a keyboard?

A keyboard is a device used to input text and commands into a computer.

5. What is a monitor?

A monitor is a device used to display the output of a computer.

6. What is a printer?

A printer is a device used to create a hard copy of a digital document.

7. What is a scanner

A scanner is a device used to convert a physical document into a digital image.

8. What is a modem?

A modem is a device used to connect a computer to the internet.

9. What is a router?

A router is a device used to direct network traffic between different devices.

10. What is a firewall?

A firewall is a security device that monitors and controls incoming and outgoing network traffic.

11. What is an operating system?

An operating system is the software that manages a computer's hardware and software resources, and provides services to applications.

12. What is a file system?

A file system is the method and data structure used by an operating system to store and organize files on a disk.

13. What is a virus?

A virus is a type of malicious software or program that can replicate itself and spread to other computers.

14. What is a backup?

A backup is a copy of important data that can be used to restore the original data in case of loss or damage.

15. What is a browser?

A browser is a software application used to access and view websites on the internet.
