Improve your Computer Knowledge-2

What is a search engine?

A search engine is a website that allows users to search for information on the internet.

What is a hyperlink?

A hyperlink is a clickable link that takes the user to another webpage or document.

What is a cookie?

A cookie is a small file that a website stores on a user's computer, which can be used to remember the user's preferences or login information.

What is a password?

A password is a secret combination of characters used to authenticate a user's identity and provide access to a computer or online account.

What is a software update?

A software update is a new version of software that contains bug fixes, security patches, and new features. Updating software can improve performance, security, and compatibility.

What is a URL?

A URL (Uniform Resource Locator) is the address of a web page on the internet.

What is a domain name?

A domain name is the unique name that identifies a website on the internet by name.

What is cloud computing?

Cloud computing is the practice of using remote servers to store, manage, and process data.

What is a domain name?

A domain name is the unique name that identifies a website on the internet by name.

What is a virtual machine?

A virtual machine is a software emulation of a computer system that can run programs and applications.

What is encryption?

Encryption is the process of encoding information to make it unreadable to anyone without the proper key or password.

What is a database?

A database is an organized collection of data that can be accessed and managed by computer software.

What is a spreadsheet?

A spreadsheet is a software program that allows users to organize and analyze data in rows and columns.

What is a word processor?

A word processor is a software program used to create, edit, and format text documents.

What is a presentation software?

Presentation software is a tool used to create multimedia presentations, such as slideshows.

What is a graphics program?

A graphics program is software used to create and edit digital images, such as photos or illustrations.

What is a video editing program?

A video editing program is software used to edit and manipulate video footage.
