Improve your Computer Knowledge-5

What is a desktop?

A desktop is the main screen or user interface of a computer.

What is a taskbar?

A taskbar is a bar at the bottom of the desktop that displays open windows and provides quick access to commonly used applications.

What is a menu bar?

A menu bar is a horizontal bar at the top of a window that displays menus for accessing different functions or options.

What is a folder?

A folder is a virtual container used to organize files and other folders on a computer.

What is a file extension?

A file extension is a set of characters added to the end of a file name to indicate the file type and the software program used to open it.

What is a hard drive?

A hard drive is a storage device used to store digital data on a computer.

What is a solid-state drive?

A solid-state drive (SSD) is a newer type of storage device that uses flash memory to store data instead of spinning disks.
